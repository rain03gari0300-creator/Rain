🔍 Proyecto de Análisis Exploratorio de Datos (EDA)

📊 Descripción del Proyecto

Este proyecto contiene dos análisis exploratorios de datos completos y profesionales:

  1. Salud del Sueño y Estilo de Vida (Clasificación)
  2. Costos de Seguro Médico (Regresión)

Cada análisis incluye preprocesamiento de datos, análisis univariado y bivariado, detección de valores atípicos, análisis de correlación, y división de datos para entrenamiento y prueba.

sleep-insurance-eda/ ├── data/ │ ├── raw/ # Coloque aquí los archivos CSV │ └── processed/ # Datos procesados y divisiones ├── reports/ │ ├── figures/ # Gráficos generados │ └── notebooks/ # Notebooks resumen ├── src/ │ ├── utils.py # Funciones auxiliares │ ├── eda_sleep.py # Pipeline EDA para sueño │ └── eda_insurance.py # Pipeline EDA para seguros ├── docs/ │ └── index.html # Esta página ├── README.md # Documentación completa ├── requirements.txt # Dependencias └── .gitignore # Archivos ignorados

🚀 Cómo Ejecutar

1. Preparar el entorno

cd sleep-insurance-eda python -m venv .venv source .venv/bin/activate # En Windows: .venv\Scripts\activate pip install -r requirements.txt

2. Agregar los datasets

Coloque los siguientes archivos CSV en data/raw/:

  • Sleep_health_and_lifestyle_dataset.csv
  • insurance.csv

3. Ejecutar los análisis

# Análisis de salud del sueño python src/eda_sleep.py # Análisis de costos de seguro médico python src/eda_insurance.py

📖 Decisiones de Análisis

Filtrado de Valores Atípicos (IQR)

Se utiliza el método IQR (Rango Intercuartílico) para detectar y remover valores atípicos. Este método es robusto y no asume una distribución normal. Los valores fuera del rango [Q1 - 1.5*IQR, Q3 + 1.5*IQR] se consideran atípicos.

Selección de Características

  • Salud del Sueño: Se elimina bp_diastolic si la correlación con bp_systolic es > 0.8 (reduce multicolinealidad)
  • Seguros: Se elimina region debido a baja correlación con el target (< 0.1)

División de Datos

  • Clasificación: División 80/20 estratificada por clase del target
  • Regresión: División 80/20 estratificada por cuantiles del target (q=5)
